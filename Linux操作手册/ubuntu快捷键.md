## 一、系统通用（必记）

- **复制**：`Ctrl + C`

- **粘贴**：`Ctrl + V`

- **剪切**：`Ctrl + X`

- **全选**：`Ctrl + A`

- **撤销**：`Ctrl + Z`

- **重做**：`Ctrl + Shift + Z` 或 `Ctrl + Y`

- **关闭窗口**：`Ctrl + W`（当前标签 / 窗口）/ `Alt + F4`（整个应用）

- **最小化窗口**：`Ctrl + Alt + 0`（数字小键盘 0）或 `Super + H`

- **最大化窗口**：`Super + ↑`

- **还原窗口**：`Super + ↓`

- **窗口左 / 右分屏**：`Super + ←` / `Super + →`

- **显示所有窗口（概览）**：`Super`（单独按一下）

- **显示桌面**：`Ctrl + Alt + D`

- **锁屏**：`Super + L`

- **注销 / 切换用户**：`Ctrl + Alt + Delete`

- **打开终端**：`Ctrl + Alt + T`

- **打开文件管理器**：`Super + E`

- **打开设置**：`Super + I`

- **打开搜索（应用 / 文件）**：`Super` 或 `Super + A`

- 截图

  ：

  - 全屏截图：`PrintScreen`
  - 选区截图：`Shift + PrintScreen`
  - 窗口截图：`Alt + PrintScreen`
  - 截图并编辑：`Shift + Super + S`

  

------

## 二、窗口与工作区

- **切换窗口**：`Alt + Tab`（按住 Alt 再按 Tab）
- **反向切换窗口**：`Alt + Shift + Tab`
- **切换同应用的多个窗口**：`Alt + `（` 是 Tab 上面那个键）
- **移动窗口到其他工作区**：`Ctrl + Alt + Shift + ←/→/↑/↓`
- **切换工作区**：`Ctrl + Alt + ←/→`
- **移动窗口到下一个显示器**：`Super + Shift + ←/→`

------

## 三、终端专用（Ubuntu Terminal）

- **复制**：`Ctrl + Shift + C`
- **粘贴**：`Ctrl + Shift + V`
- **新建标签页**：`Ctrl + Shift + T`
- **关闭标签页**：`Ctrl + Shift + W`
- **新建窗口**：`Ctrl + Shift + N`
- **清屏**：`Ctrl + L`
- **中断命令**：`Ctrl + C`
- **暂停命令**：`Ctrl + Z`
- **历史命令上 / 下**：`↑` / `↓`
- **自动补全**：`Tab`

------

## 四、Vim 编辑器（命令模式）

- **复制一行**：`yy`
- **复制 n 行**：`n yy`（如 `5yy`）
- **复制选中块**：`v` 选中 → `y`
- **粘贴**：`p`（光标后）/ `P`（光标前）
- **剪切一行**：`dd`
- **剪切 n 行**：`n dd`
- **撤销**：`u`
- **重做**：`Ctrl + R`

------

## 五、自定义快捷键（自己改）

1. 打开：`设置 → 键盘 → 键盘快捷键`
2. 找到对应功能，点击快捷键，按你想要的组合即可。
