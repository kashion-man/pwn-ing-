# 三、内存

![image-20260119144244492](./assets/image-20260119144244492.png)

**32位内存布局**

![image-20260119144423205](./assets/image-20260119144423205.png)

​	经典布局已经被淘汰了，原因：

**其中内存映射区和栈相对增长，导致堆区内存极大可能溢出至内存映射区**

​	两者布局区别：

**经典布局中栈区和内存映射区相对增长，而现代布局是堆区和内存映射区相对增长**
